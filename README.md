# Maze Game with Pathfinding (Python)

This project is a grid-based maze game built using Python and the Pygame library. The player navigates through a procedurally generated maze while avoiding an enemy with dynamic behaviour.


Key Features:

- Random maze generation using graph traversal
- Enemy pathfinding implemented with Dijkstra’s algorithm
- Multiple difficulty modes with different AI behaviours
- Additional mechanics such as collectibles and adaptive difficulty



Focuses: 

- Structuring the program using classes and modular components
- Applying pathfinding algorithms (DFS, Dijkstra) in a dynamic environment
- Designing gameplay mechanics with consideration of user interaction and gameplay experience


How to run:

Install pygame: pip install pygame
Run the Python file
